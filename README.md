# QA Analysis Portfolio - Logistics Delivery App

## 🎯 Project Overview
This project contains a comprehensive suite of **Bug Reports** and **Functional Testing** documentation for a real-world logistics application. I focused on identifying gaps between business requirements and the actual system implementation.

## 🛠 Skills Applied
* **Manual Testing:** End-to-end testing of user flows (Drivers & Admins).
* **Bug Reporting:** Detailed documentation with Steps to Reproduce, Expected vs. Actual results, and Severity levels.
* **Requirement Analysis:** Validating software against technical specifications (Case 3.13 - Routing & Delivery).
* **AI-Assisted QA:** Utilizing AI tools to streamline test case documentation and technical English translation.

## 📑 Featured Bug Reports
I have documented **12 critical gaps and bugs**, including:
* **Routing Logic Failures:** Missing implementation of business rules for delivery exchanges/losses.
* **UI/UX Inconsistencies:** Calendar and navigation button discrepancies.
* **Workflow Gaps:** Missing status updates for "Delivery Not Realized" flows.
